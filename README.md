# togovar-stanza

[Togovar-Stanza](https://togovar.biosciencedbc.jp/stanza "Togovar-Stanza")

## Prerequisites

* Node: 14.16
* yarn: 1.22+

## Getting Started

Create a new stanza:

```
$ yarn generate [options] [id]
```

Serve the repository locally:

```
$ yarn run
```

Build stanzas for deployment:

```
$ yarn build
```
